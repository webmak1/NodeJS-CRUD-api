# NODEJS-CRUD-API

```
# Routes
GET      /api/users
POST     /api/users
GET      /api/users/${id}
PUT      /api/users/${id}
DELETE   /api/users/${id}

```

## Usage

```
# Install dependencies
npm install

# Run in develpment
npm run start:dev

# Run in production
npm run start:prod

# Run the cluster
npm run start:multi

# Run tests
npm run test
```

Server is running on port 4000
